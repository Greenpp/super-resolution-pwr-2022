# Super-resolution

## How to get data

```sh
dvc repro
```

## How to contribute

### Tools

[Poetry](https://python-poetry.org/)

[Pre-commit](https://pre-commit.com/)

[Pyenv](https://github.com/pyenv/pyenv)

### Steps

1. Clone repository

2. Install pre-commit hooks

   ```sh
   pre-commit install
   ```

3. Install dependencies

   ```sh
   poetry install
   ```
